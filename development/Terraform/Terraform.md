# Terraform

Terraform es una herramienta de Infrastructure as Code (IaC) desarrollada por HashiCorp que te permite definir, provisionar y gestionar infraestructura de forma automatizada usando código. En lugar de configurar manualmente servidores, redes u otros recursos, con Terraform escribes archivos de configuración que describen cómo debe ser tu infraestructura, y luego Terraform se encarga de crearla, modificarla o destruirla en la nube o localmente según ese código.

### ¿Qué puede hacer terraform?

* Crear servidores en AWS, Azure, GCP, etc.
* Configurar redes, balanceadores, bases de datos
* Desplegar clusters de Kubernetes
* Automatizar cambios y destrucción de recursos

## ¿Cómo funciona terraform?

Terraform crea y gestiona recursos en plataformas en la nube y otros servicios mediante sus interfaces de programación de aplicaciones (API).
